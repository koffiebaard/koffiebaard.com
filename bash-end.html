Bash-end

A back-end written solely in Bash with only one (1) dependency. It supports CRUD operations, and hooks up directly to nginx through fastcgi. It's easy to use, fast and efficient.

Features:

  • Efficient routing
  • Easily built controllers and models
  • Powerful input validation system
  • Only one dependency; jq
  • Decent test coverage
  • Out of the box MySQL support
